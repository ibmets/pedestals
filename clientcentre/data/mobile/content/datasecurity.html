Bluejot

The Bluejot app for iOS and Android provides the ability for IBM Employees to store notes securely.

Note text and images are stored in an encrypted local database to ensure that confidential data is not recoverable even in the event of a device being stolen.

Bluejot has replication capabilities which enable the locally stored secure data to be mirrored across platforms to any device that has the bluejot app installed.

Secure Browser

The IBM Secure Browser app for iOS and Android enables employees to securely access corporate Internet sites and networks with no VPN required. It was written as a prototype for IBM Security and proved the feasibility of the concept.
