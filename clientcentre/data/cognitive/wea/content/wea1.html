Watson Engagement Advisor can answer questions in natural language for a given domain with sufficient training. But it is not well suited to handle information that can be sourced directly from existing structured sources like database tables.

Hudson is a new component that sits alongside WEA, that uses a fixed model of the domain in question with existing data sources mapped to this model.

It represents the model and data in a structured form of English that means it is possible to map questions posed in everyday normal English on to its structured English model of the world, connecting questions to data sources and ultimately the answer.
