Variable width Sidescroller items

	.overthrow-enabled .sidescroll ul {
	width: 800%;
}
@media (min-width: 40em){
	.overthrow-enabled .sidescroll ul {
		width: 400%;
	}
}
@media (min-width: 60em){
	.overthrow-enabled .sidescroll ul {
		width: 200%;
	}
}

One Sidescroller item is shown at a time given a browser window smaller than 40em. Eight items occupying a container set to 800% of the available Sidescroller “viewport,” leaving one visible at a time.

Two Sidescroller items are shown at a time given a browser window smaller than than 60em. Eight items occupying a container set to 400% of the available Sidescroller “viewport,” leaving two visible at a time.

Four items are shown at a time given a browser window larger than 60em. Eight itemsm occupying a container set to 200% of the available Sidescroller “viewport,” leaving four visible at a time.
